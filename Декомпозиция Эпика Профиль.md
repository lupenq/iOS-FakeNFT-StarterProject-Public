 Критерии готовности эпика
 Главный экран профиля и редактирование
  Экран профиля корректно отображает аватар, имя, описание, сайт и количество NFT в списках.
  Сайт открывается внутри приложения через WKWebView.
  Экран редактирования открывается модально; измененные данные отправляются на сервер и обновляют главный экран профиля.
  По кнопке «Изменить фото» появляется алерт для ввода ссылки на изображение.Экран «Мои NFT»
  UITableView отображает купленные NFT пользователя с картинкой, названием, автором и ценой.
  Сортировка работает по всем трем критериям (цена, рейтинг, название) и сохраняется в UserDefaults.
  Если у пользователя нет NFT, показывается плейсхолдер «У Вас еще нет NFT».Экран «Избранное»
  UICollectionView отображает NFT, лайкнутые пользователем, в две колонки.
  Нажатие на кнопку-сердце удаляет NFT из списка избранного и отправляет запрос на сервер.
  Если список лайков пуст, показывается плейсхолдер «У Вас еще нет избранных NFT».
